Para este proyecto llamado UnitConverter la idea es crear un programa que convierta unidades de medida y, además, practicar todo el proceso de usar Git desde la terminal.

En el repositorio nuevo que hemos creado hay:

Un archivo Java con el conversor básico. Otro archivo con la versión mejorada para la rama feature-update. Una pagina HTML sencilla para GitHub Pages. Una hoja con comandos de Git. Una plantilla para hacer la presentación. También explica cómo compilar y ejecutar el programa con javac y java y como probar la versión de la rama nueva y el flujo de Git: crear el repo, hacer commits, crear una rama, fusionarla y subirlo a GitHub. Y por último dice cómo activar GitHub Pages y que en la presentación hay que poner capturas de los comandos usados
